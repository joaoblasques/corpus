# lakeflow-a-new-era-of-agentic-data-engineering

> **Quick intake** (web). [open source]("https://www.databricks.com/blog/lakeflow-new-era-agentic-data-engineering")

Databricks Lakeflow is a unified platform for data engineering that offers significant advantages over complex and fragmented data stacks. It provides a unified architecture for ingestion, transformation, and orchestration, enabling agents to build and operate data pipelines. Lakeflow includes features such as Genie Code, Lakeflow Designer, Genie ZeroOps, and Lakeflow Connect.
