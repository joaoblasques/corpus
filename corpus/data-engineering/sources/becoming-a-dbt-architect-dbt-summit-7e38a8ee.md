# Becoming a dbt Architect | dbt Summit

> **Quick intake** (obsidian-list). [open source](https://www.getdbt.com/dbt-summit/sessions/becoming-a-dbt-architect)

This course covers practical architecture patterns for dbt implementations, including environment strategies, project structure, and operational workflows. It aligns with the dbt Architect certification and is designed for platform leaders and technical owners. The course covers key security and governance controls for collaboration.

**Key topics**
- dbt Architect certification
- dbt implementation patterns
- environment strategies
- project structure
- operational workflows
- security and governance controls
