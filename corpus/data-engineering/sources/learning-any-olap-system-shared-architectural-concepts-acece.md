# "Learning Any OLAP System: Shared Architectural Concepts"

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-olap-systems-learning-framework-shared-architecture.md`

The document discusses the shared architectural concepts of OLAP systems, enabling faster learning and transfer of knowledge between systems. It highlights the differences between shared-nothing and shared-disk architectures and the importance of metadata management. Understanding these fundamentals provides an 'unfair advantage' when migrating between platforms.

**Key topics**
- Shared-nothing architecture
- Shared-disk architecture
- Metadata management
- OLAP systems
