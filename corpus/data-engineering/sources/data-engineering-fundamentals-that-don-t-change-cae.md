# Data Engineering Fundamentals That Don't Change

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-data-engineering-fundamentals-that-dont-change.md`

A concise list of data engineering fundamentals that remain relevant despite tool and platform changes. These durable foundations include concepts like ANSI SQL, CAP theorem, and idempotent pipelines. Mastering these concepts is essential for data engineers.

**Key topics**
- ANSI SQL
- CAP theorem
- OLTP vs OLAP
- Job orchestration
- Fact data modeling
- Data quality testing
