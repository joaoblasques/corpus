# data-engineering-portfolio-5-projects-that-get-hired

> **Quick intake** (web). [open source]("https://www.dataexpert.io/blog/data-engineering-portfolio-projects-get-hired")

This article highlights five key projects that hiring managers value most in data engineering, covering skills like real-time streaming, ETL workflows, data modeling, and observability.

**Key topics**
- Real-Time Streaming with Kafka & Snowflake
- ETL with Airflow & Databricks
- Data Warehouse with Snowflake & dbt
- Data Lake on AWS S3 & Redshift
- Pipeline with Observability (PySpark & Monte Carlo)
