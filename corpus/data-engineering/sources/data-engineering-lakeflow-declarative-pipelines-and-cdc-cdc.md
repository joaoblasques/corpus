# Data Engineering - LakeFlow Declarative Pipelines and CDC

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-data-engineering-lakeflow-declarative-pipelines-and-cdc.md`

LakeFlow Declarative Pipelines simplify change data capture (CDC) using declarative SQL, reducing code complexity and handling edge cases. This approach separates business logic from failure handling and provides automatic optimization and error recovery. Key features include simplified authoring, intelligent operations at scale, and unified streaming and batch processing.

**Key topics**
- LakeFlow Declarative Pipelines
- Change Data Capture (CDC)
- Declarative SQL
