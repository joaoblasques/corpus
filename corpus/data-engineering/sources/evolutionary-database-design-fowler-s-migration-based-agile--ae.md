# Evolutionary Database Design — Fowler's Migration-Based Agile DB Patterns

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-evolutionary-database-design-fowler-migrations-agile.md`

The document discusses Fowler's Migration-Based Agile DB Patterns for evolving databases continuously alongside application code in agile environments. It outlines core practices for safe and automated database evolution, including version control, migration-based changes, and collaboration between DBAs and developers. The result is a database that evolves safely at the same pace as the application.
