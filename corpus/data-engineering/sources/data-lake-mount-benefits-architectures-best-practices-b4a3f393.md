# "Data Lake Mount: Benefits, Architectures & Best Practices"

> **Quick intake** (obsidian-list). [open source](https://lakefs.io/blog/data-lake-mount/)

Data lake mounts provide a unified filesystem namespace for accessing distributed data, simplifying pipeline development and centralizing access controls. They offer consistent, secure access to raw and curated datasets while separating storage and computing. Data lake mounts also introduce considerations for credential rotation, mount lifecycle management, and potential caching or consistency semantics.

**Key topics**
- Data Lake Mounts
- Unified Filesystem Namespace
- Pipeline Development
- Access Controls
- Credential Rotation
- Mount Lifecycle Management
