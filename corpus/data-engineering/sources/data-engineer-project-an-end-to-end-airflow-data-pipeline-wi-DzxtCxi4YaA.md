# "Data Engineer Project: An end-to-end Airflow data pipeline with BigQuery, dbt Soda, and more!"

> **Quick intake** (YouTube · Data with Marc · playlist _Data Engineering_). [watch on YouTube](https://youtu.be/DzxtCxi4YaA) · [transcript](../../../raw/youtube/youtube-DzxtCxi4YaA-data-engineer-project-an-end-to-end-airflow-data-pipeline-wi.md)

This video covers building an end-to-end Airflow data pipeline with BigQuery, dbt, and Soda. It includes loading a CSV file into Google Cloud Storage, creating a BigQuery dataset, and uploading the CSV file into a BigQuery table. The video also covers data quality checks and creating a dashboard with Metabase.
