# BI as Code — SQL and Markdown Dashboards for Data Engineers

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-bi-as-code-sql-markdown-dashboards.md`

The article discusses the benefits of treating dashboards as code, written in SQL and Markdown, to eliminate vendor lock-in and improve reproducibility. This approach, known as BI as Code, mirrors how modern data teams manage transformations and orchestration. It is particularly suited for teams already writing SQL and using Git.
