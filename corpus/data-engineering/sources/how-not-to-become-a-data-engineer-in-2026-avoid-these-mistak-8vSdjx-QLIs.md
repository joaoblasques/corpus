# How NOT to Become a Data Engineer in 2026 (Avoid These Mistakes)

> **Quick intake** (YouTube · Data with Baraa · playlist _Corpus_queue_). [watch on YouTube](https://youtu.be/8vSdjx-QLIs) · [transcript](../../../raw/youtube/youtube-8vSdjx-QLIs-how-not-to-become-a-data-engineer-in-2026-avoid-these-mistak.md)

The video discusses common mistakes that prevent individuals from becoming data engineers, including trying to learn every tool, focusing on tools over data engineering skills, and ignoring SQL. The key takeaway is to focus on one stack, learn data engineering concepts, and prioritize SQL skills.

**Key topics**
- Mistakes in becoming a data engineer
- Importance of focusing on one stack
- Data engineering concepts
- SQL skills
- Data engineering tools
- Avoiding technology overload
