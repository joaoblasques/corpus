# Realtime Data Streaming |  End To End Data Engineering Project

> **Quick intake** (YouTube · CodeWithYu · playlist _Data Engineering_). [watch on YouTube](https://youtu.be/GqAcTrqKcrY) · [transcript](../../../raw/youtube/youtube-GqAcTrqKcrY-realtime-data-streaming-end-to-end-data-engineering-project.md)

This video covers an end-to-end data engineering project that utilizes Apache Airflow, Apache Zookeeper, Kafka, Cassandra, and Postgres SQL, all containerized with Docker. The project fetches data from a random user API and streams it into Kafka, which is then visualized in a control center. The data is also streamed into Cassandra using Apache Spark.

**Key topics**
- Apache Airflow
- Apache Zookeeper
- Kafka
- Cassandra
- Postgres SQL
- Docker
