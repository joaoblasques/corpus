# Guitar - Ultimate Fretboard System Rectangle and Stack

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-guitar-ultimate-fretboard-system-rectangle-and-stack.md`

This document describes a system for visualizing and navigating the fretboard of a guitar, allowing for the creation of various scales and arpeggios. The system is based on two shapes, rectangles and stacks, which contain the pentatonic scale and can be used to derive other scales and modes. The system provides a hierarchical approach to music theory, prioritizing chord tones and modal add-notes.
