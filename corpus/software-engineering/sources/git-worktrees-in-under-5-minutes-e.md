> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-git-worktrees-in-under-5-minutes.md`

This document explains Git worktrees, a feature that allows multiple branches to be checked out simultaneously without cloning the repository multiple times. It covers the key behavior, commands, and a typical workflow. The key takeaway is that Git worktrees enable parallel development and isolation of tasks.
