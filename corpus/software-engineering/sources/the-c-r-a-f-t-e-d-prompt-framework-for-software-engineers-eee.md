# The C.R.A.F.T.E.D. Prompt Framework for Software Engineers

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-the-c-r-a-f-t-e-d-prompt-framework-for-software-engineers.md`

The C.R.A.F.T.E.D. Prompt Framework is a structured method for constructing high-quality prompts to get optimal results from AI models for software engineering tasks. It provides a logical workflow that mirrors how developers approach problems, emphasizing the importance of context, role, action, format, tone, examples, and definition of done. The framework is designed to be flexible and adaptable to different software engineering tasks and workflows.
