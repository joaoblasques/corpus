# Learning Algorithm (PDF, 327 pages)

**TL;DR**: A compact algorithms reference PDF (327 pages, 9 parts) covering greedy scheduling, graph shortest-path algorithms (BFS/Dijkstra/Floyd-Warshall), dynamic programming, divide-and-conquer sorting, and string matching (KMP). Author unknown; structured as a CS curriculum companion.

## Coverage

- **Part 2**: Interval scheduling — earliest-finish-time greedy; exchange-argument correctness proof.
- **Part 3**: Divide and conquer fundamentals; recurrence relations.
- **Part 4**: BFS for unweighted shortest paths; Dijkstra's algorithm; why BFS double-counts in weighted graphs.
- **Part 5**: Floyd-Warshall all-pairs shortest path; negative cycle detection via main-diagonal check.
- **Part 6**: Dynamic programming (coin change, knapsack variants).
- **Part 7**: Merge sort implementation (C/pseudocode).
- **Part 8**: Quicksort — Hoare vs. Lomuto partition; Hoare's three-times-fewer-swaps advantage.
- **Part 9**: Knuth-Morris-Pratt (KMP) string matching; failure function construction and matching phase.
