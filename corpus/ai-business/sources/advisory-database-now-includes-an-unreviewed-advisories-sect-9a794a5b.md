# Advisory Database now includes an Unreviewed Advisories section - GitHub Changelog

> **Quick intake** (obsidian-list). [open source](https://github.blog/changelog/2021-12-16-advisory-database-now-includes-an-unreviewed-advisories-section)

The GitHub Advisory Database now includes an Unreviewed Advisories section, which auto-publishes unreviewed advisories from the National Vulnerability Database feed. This allows users to find more advisories, but also preserves the curated quality of Dependabot alerts for reviewed advisories. Users can filter advisories by type.

**Key topics**
- GitHub Advisory Database
- Unreviewed Advisories
- National Vulnerability Database
- Dependabot alerts
- GitHub Reviewed advisories
