# "Spec-driven development with AI: Get started with a new open source toolkit - The GitHub Blog"

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-spec-driven-development-with-ai-open-source-toolkit.md`

Spec-driven development is a methodology that prioritizes creating detailed specifications before coding begins, ensuring AI tools can effectively execute tasks. This approach transforms specifications into living, executable artifacts that guide the development process, enhancing reliability and quality in coding projects. It is particularly valuable as AI coding agents become prevalent in software engineering.

- Spec-driven development

- Methodology
- Specifications
- GitHub Spec Kit
