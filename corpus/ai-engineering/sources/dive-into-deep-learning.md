# Dive into Deep Learning (D2L)

**TL;DR**: Comprehensive, code-first deep learning textbook (1151 pages, CC BY-SA 4.0) by Aston Zhang, Zachary C. Lipton, Mu Li, and Alexander J. Smola. Free at d2l.ai. Every concept ships with runnable PyTorch (and JAX/MXNet) code in interactive Jupyter notebooks. Spans from linear regression through transformers, RNNs, attention, NLP pretraining, computer vision, optimization, and computational performance [^src1].

## What it is

"Dive into Deep Learning" is the standard open textbook for the field. Key design choices [^src1][^src2]:
- **Code-in-the-book**: every algorithm is implemented from scratch before the high-level API version appears; the principle is to understand what the framework is doing under the hood before using it
- **Interactive Jupyter notebooks**: each section is a live notebook; readers run and modify the code, not just read about it
- **Multiple frameworks**: PyTorch is primary; JAX and MXNet versions exist
- **Math + implementation**: derivations are paired with implementations — you see the gradient formula, then the Python that computes it
- **Pedagogically sequenced**: shallow networks before deep ones; from-scratch before concise; concepts build strictly

The authors describe the book's goal as bridging "the gap between theoretical foundations and practical applications" [^src1].

## Key pedagogical approach

The motivating example in Chapter 1 is a smartphone voice recognition pipeline (Hey Siri / OK Google / wake word detection) — illustrating that machine learning is "programming with data" rather than explicit rule-writing [^src2]. The book defines the four key components of any ML system: **data**, **model**, **objective function**, and **optimization algorithm** [^src2].

For linear regression, the book shows the analytic solution (normal equations: w* = (X^T X)^{-1} X^T y) before introducing gradient descent, making explicit why iterative methods are necessary for deep networks [^src6].
