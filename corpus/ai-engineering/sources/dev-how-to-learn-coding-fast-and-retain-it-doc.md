# Dev - How to Learn Coding Fast and Retain It

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-dev-how-to-learn-coding-fast-and-retain-it.md`

A system for learning programming and retaining it, demonstrated by learning computer vision. The system involves setting specific goals, creating a roadmap, challenging oneself, and reviewing with spaced repetition. It emphasizes building a deep connected understanding of concepts.

**Key topics**
- Learn → Practice → Apply → Review
- Spaced Repetition
- Building a Knowledge Web
- Daily Habit
- Pattern Recognition
- Machine Learning
