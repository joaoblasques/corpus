# Building a multimodal lakehouse for AI (w/ Chang She)

> **Quick intake** (obsidian-list). [open source](https://roundup.getdbt.com/p/building-a-multimodal-lakehouse-for)

Chang She discusses building a multimodal lakehouse for AI, leveraging the Lance file format and LanceDB, a data platform for large-scale AI production. He highlights the importance of multimodal data and the need for a new infrastructure layer for AI-native data.

**Key topics**
- Lance file format
- LanceDB
- Multimodal lakehouse
- AI-native data platform
