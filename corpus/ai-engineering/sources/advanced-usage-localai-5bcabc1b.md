# "Advanced usage :: LocalAI"

> **Quick intake** (obsidian-list). [open source](https://localai.io/advanced/index.html)

LocalAI is a tool for model configuration and management. It uses YAML files to define model parameters and behavior, and supports automatic prompt caching and connecting external backends. LocalAI can also be extended with extra backends and supports parallel requests.

**Key topics**
- LocalAI
- Model Configuration
- YAML Files
- Prompt Templates
- API Endpoints
- Model Gallery
