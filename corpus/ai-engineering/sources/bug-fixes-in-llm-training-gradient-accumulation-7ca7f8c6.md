# Bug Fixes in LLM Training - Gradient Accumulation

> **Quick intake** (obsidian-list). [open source](https://unsloth.ai/blog/gradient)

The document discusses a bug in LLM training using gradient accumulation, which leads to a higher loss than full batch training. Researchers found that the issue is caused by the denominator in the cross-entropy loss calculation. A new methodology has been developed to fix the issue, which is now available in Unsloth.
