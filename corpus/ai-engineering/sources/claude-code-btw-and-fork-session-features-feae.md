# Claude Code - btw and fork-session Features

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-claude-code-btw-and-fork-session-features.md`

Claude Code introduces two new features: /btw for quick side questions and /fork-session for deep dives. These features help maintain a clean context window and improve LLM performance. The /btw feature provides ephemeral answers, while /fork-session allows for multi-turn investigations.
