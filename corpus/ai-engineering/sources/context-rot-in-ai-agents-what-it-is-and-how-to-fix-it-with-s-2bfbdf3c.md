# "Context Rot in AI Agents: What It Is and How to Fix It with Session Handoffs"

> **Quick intake** (obsidian-list). [open source](https://www.mindstudio.ai/blog/context-rot-ai-agents-session-handoff-fix-2/)

Context rot in AI agents occurs when they forget previous context, leading to degraded performance. Session handoffs can fix this by clearing context without losing progress. Techniques like Claude Code's CLAUDE.md file help manage context.
