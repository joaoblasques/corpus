# Music Tech - Dubler 2 Voice-to-MIDI Chord Progressions

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-music-tech-dubler-2-voice-to-midi-chord-progressions.md`

Dubler 2 is a software that converts voice into MIDI control, allowing users to create drum patterns, melodies, and chord progressions without touching a keyboard. It uses a training process to learn vocal patterns and can be connected to a DAW for further editing. The software is particularly useful for non-keyboard players and live performance trigger control.

- Dubler 2

- MIDI control
- Chord progressions
