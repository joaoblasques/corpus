# Benchmarking Single Agent Performance

> **Quick intake** (obsidian-list). [open source](https://www.langchain.com/blog/react-agent-benchmarking)

This document benchmarks the performance of a single ReAct agent with varying context sizes, evaluating its ability to perform tasks in Calendar Scheduling and Customer Support domains. The results show that more context and tools degrade agent performance, and agents that require longer trajectories degrade more quickly. The best-performing models are o1, o3-mini, and claude-3.5-sonnet.
