# Claude Cowork - Getting Started Guide

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-claude-cowork-getting-started-guide.md`

Claude Cowork is a non-developer accessible version of Claude Code, allowing users to read and write local files, browse the web, and produce polished deliverables without manual uploads/downloads. It has key capabilities such as direct local file access, web browsing, and multi-step sub-agent coordination. However, it has limitations like heavy token consumption and no memory across sessions.

**Key topics**
- Claude Cowork
- Claude Code
- Local File Access
- Web Browsing
- Sub-Agent Coordination
- Polished Deliverables
