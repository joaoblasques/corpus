# backbone-fundamentals

> **Quick intake** (obsidian-list). [open source](https://addyosmani.com/backbone-fundamentals/)

This book covers the fundamentals of Backbone.js, a lightweight JavaScript library that adds structure to client-side code. It teaches developers how to build maintainable and scalable web applications using Backbone.js, focusing on its Model-View-Controller (MVC) pattern and event-driven architecture. The book also explores modern JavaScript practices and how Backbone.js influences modern frameworks.

**Key topics**
- Backbone.js
- Model-View-Controller (MVC) pattern
- Event-driven architecture
- JavaScript
- Web application development
- Client-side code structure
