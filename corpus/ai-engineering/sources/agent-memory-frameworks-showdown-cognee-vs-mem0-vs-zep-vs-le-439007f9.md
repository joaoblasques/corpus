> **Quick intake** (obsidian-list). [open source](https://theaiengineer.substack.com/p/cognee-vs-zep-vs-mem0-vs-letta)

The document compares four memory frameworks for AI agents: Mem0, Zep, cognee, and Letta. It discusses their strengths and weaknesses, including their memory models, write paths, and adoption rates. The key takeaway is that memory frameworks are not winning on accuracy, but rather on reducing response time and token usage.
