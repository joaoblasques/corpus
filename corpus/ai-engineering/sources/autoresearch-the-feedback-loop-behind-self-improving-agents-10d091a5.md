# "Autoresearch: The feedback loop behind self-improving agents"

> **Quick intake** (obsidian-list). [open source](https://www.latent.space/p/autoresearch-introspection)

Autoresearch is a feedback loop mechanism for self-improving agents, allowing them to maintain and improve primary systems using feedback signals, evaluations, and human input. Introspection, a company building infrastructure for deploying these systems, presents three patterns for autoresearch in production: the loop as the product, agent recipes, and optimizing for progress. The key takeaway is that humans remain central to the software factory.
