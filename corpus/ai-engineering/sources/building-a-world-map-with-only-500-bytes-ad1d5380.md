# Building a World Map with only 500 bytes

> **Quick intake** (obsidian-list). [open source](https://simonwillison.net/2026/Jul/4/building-a-world-map-with-only-500-bytes/#atom-everything)

A method to generate a credible ASCII world map using 445 bytes of data through deflate compression and JavaScript. The technique utilizes fetch() with data URIs and DecompressionStream. This approach enables efficient map generation.

**Key topics**
- ASCII world map
- deflate compression
- JavaScript
- fetch() with data URIs
- DecompressionStream
- efficient map generation
