# How DeepSeek Runs a 284B LLM on a Laptop (Run AI Locally)

> **Quick intake** (YouTube · The Stack · playlist _Corpus_queue_). [watch on YouTube](https://youtu.be/3gk-9p9DSGs) · [transcript](../../../raw/youtube/youtube-3gk-9p9DSGs-how-deepseek-runs-a-284b-llm-on-a-laptop-run-ai-locally.md)

The video discusses how DeepSeek's 284B LLM was run on a laptop using a pure-C engine, achieving 26 tokens per second. This was made possible by a Mixture-of-Experts model and asymmetric quantization. The creator of Redis built an inference engine that runs the model locally, with a validated weight file and a guarantee of correctness.
