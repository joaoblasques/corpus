# Claude Code - Playwright MCP for CSS and Frontend Debugging

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-claude-code-playwright-mcp-for-css-and-frontend-debugging.md`

Playwright MCP is a debugging tool that allows for CSS and frontend debugging by executing JavaScript and inspecting live CSS being applied in the browser. It helps in identifying the root cause of issues by providing a process of elimination. The tool is particularly useful when CSS is the issue.

**Key topics**
- Playwright MCP
- CSS debugging
- Frontend debugging
- JavaScript execution
- Live CSS inspection
- Root cause analysis
