# 12. Virtual Environments and Packages

> **Quick intake** (obsidian-list). [open source](https://docs.python.org/3/tutorial/venv.html)

Virtual environments and packages are used to manage Python applications with conflicting requirements. A virtual environment is a self-contained directory tree that contains a Python installation and additional packages. This allows different applications to use different versions of packages without conflicts.
