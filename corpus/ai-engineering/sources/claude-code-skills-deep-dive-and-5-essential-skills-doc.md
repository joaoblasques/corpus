# Claude Code - Skills Deep Dive and 5 Essential Skills

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-claude-code-skills-deep-dive-and-5-essential-skills.md`

Claude Code is a system that can be specialized with skills to perform specific tasks, transforming it from a general assistant. These skills are like 'plugging a USB into Claude's head' and can be used to bypass the ceiling of AI wrapper tools. The compounding value increases as more skills are built and they learn from feedback.
