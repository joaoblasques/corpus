# "Claude Tag for Slack: How Anthropic's Native Agent Changes Enterprise Workflows"

> **Quick intake** (obsidian-list). [open source](https://www.mindstudio.ai/blog/claude-tag-slack-enterprise-agent/)

Claude Tag for Slack is a native integration that brings AI into the flow of work, allowing teams to tag Claude in conversations and receive contextually relevant responses. It can summarize threads, draft messages, and provide research and explanation, making it a valuable tool for enterprise teams. However, it has limitations, such as no live web access and no persistent memory across channels.
