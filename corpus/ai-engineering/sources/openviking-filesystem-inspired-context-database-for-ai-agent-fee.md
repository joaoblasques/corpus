# OpenViking — Filesystem-Inspired Context Database for AI Agents

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-openviking-context-database-ai-agents-filesystem.md`

OpenViking is an open-source context database that addresses the problem of fragmented context in long-running AI agents. It uses a filesystem-inspired paradigm with a three-tier hierarchical loading structure to reduce token consumption and enable precise retrieval. This unified management of memories, resources, and skills enables AI agents to improve through use.
