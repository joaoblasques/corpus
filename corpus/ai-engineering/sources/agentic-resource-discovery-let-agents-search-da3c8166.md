# "Agentic Resource Discovery: Let agents search"

> **Quick intake** (obsidian-list). [open source](https://huggingface.co/blog/agentic-resource-discovery-launch)

Agentic Resource Discovery (ARD) is a specification for discovering and indexing agent capabilities. It separates discovery from execution and allows for dynamic search and retrieval of capabilities. The Hugging Face Discover Tool is a reference implementation of ARD.
