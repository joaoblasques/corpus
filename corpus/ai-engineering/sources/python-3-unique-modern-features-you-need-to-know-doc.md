# Python - 3 Unique Modern Features You Need to Know

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-python-3-unique-modern-features-you-need-to-know.md`

This document covers three modern Python features: the match statement, the walrus operator, and type hints with TypeVar and Generic. These features improve code readability and reduce repetition. The match statement is a powerful alternative to if/elif chains, the walrus operator simplifies assignments and checks, and modern type hints make code more readable.

**Key topics**
- match statement
- walrus operator
- type hints
- TypeVar
- Generic
- Python 3.10+
