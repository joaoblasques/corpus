# Beta SDKs for the 2026-07-28 MCP Spec Release Candidate Are Here

> **Quick intake** (obsidian-list). [open source](https://blog.modelcontextprotocol.io/posts/sdk-betas-2026-07-28/)

The MCP protocol is undergoing a major revision, going stateless and removing the initialize handshake and protocol-level session. This change enables server scaling using a simple round-robin load balancer and introduces new patterns for server-to-client interactions. The beta SDKs for the 2026-07-28 MCP Spec Release Candidate are now available for testing.

- MCP protocol revision
