# "Data Agents: Build Reliable Enterprise AI on Trusted Data"

> **Quick intake** (obsidian-list). [open source](https://lakefs.io/blog/data-agents/)

Data agents are AI-enabled systems that automate data collection, processing, and analysis, enabling enterprises to accelerate analytics and decision-making. They require a trusted data foundation and can be built using software engineering principles. Data versioning and reproducibility are crucial for agent workflows.
