# Probabilistic Machine Learning: An Introduction (Murphy, 2022)

TL;DR: Modern probabilistic ML textbook by Kevin P. Murphy (MIT Press, 2022; CC-BY-NC-ND). Successor to his 2012 "Machine Learning: A Probabilistic Perspective." Covers supervised/unsupervised learning, deep neural networks, nonparametric methods, and beyond — 860 pages, Python throughout (JAX, PyTorch, scikit-learn). Full 43-part PDF ingested. See [Kevin P. Murphy](/ai-engineering/kevin-murphy.md).

## Key ideas

- **Probabilistic framing**: "all unknown quantities are random variables endowed with probability distributions." This makes ML methods interoperate with stochastic optimization, control theory, information theory, and statistical physics. [^pml-p01]
- **Classification fundamentals** (Chapter 1): a decision tree of depth 2 can separate Iris flowers using only petal-length and petal-width; empirical risk minimization defines model fitting; generalization — not training accuracy — is the actual goal. [^pml-p01]
- **Bayesian decision theory** (Chapter 5): zero-one loss → MAP estimate = argmax posterior; ROC curves plot TPR vs FPR as threshold τ varies; AUC summarizes as single scalar; Bayesian Occam's razor — marginal likelihood inherently prefers simpler models via conservation of probability mass. [^pml-p10]
- **Model selection criteria** (Chapter 5): BIC ≈ log p(D|θ̂) − (Dm/2)log N; AIC = log p(D|θ̂) − Dm (less penalty than BIC); MDL = same form as BIC via two-part coding; WAIC replaces MAP with ELPD (expected log pointwise predictive density) + variance-based complexity — works for singular statistical models. [^pml-p10]
- **Change of variables** (Chapter 2): for an invertible function y = f(x), p(y) = p(x)|dx/dy|; the multivariate generalization uses the absolute Jacobian determinant. [^pml-p05]
- **Jacobian VJP/JVP** (Chapter 7): VJP (reverse mode) more efficient when outputs ≤ inputs; JVP (forward mode) more efficient otherwise — the theoretical basis for backpropagation via reverse-mode autodiff. [^pml-p15]
- **SGD/LMS** (Chapter 8): minibatch-1 SGD on squared-error loss = LMS/delta rule/Widrow-Hoff rule; does not decrease monotonically but converges to least-squares solution. [^pml-p16]
- **Deep learning revolution** (Chapter 13): AlexNet (2012) win on ImageNet demonstrated GPU-backed DNNs dramatically outperform handcrafted systems; MLP-mixer shows unstructured models can match CNNs on vision with massive datasets. [^pml-p22]
- **Kernel SVM** (Chapter 17): OVR (one-vs-rest) can produce ambiguous labeling and class-imbalance problems; OVO (all-pairs) requires O(C²) classifiers; C and γ (RBF bandwidth) are tightly coupled — libsvm recommends 2D grid search. [^pml-p30]
- **Graph representation learning** (Chapter 23): GraphEDM framework unifies GRL methods as encoder-decoder; transductive (fixed graph) vs inductive (generalize to new graphs); non-Euclidean spaces (hyperbolic for trees) are a growing direction. [^pml-p38]
- Python-first (JAX/PyTorch/scikit-learn); all figures reproducible via linked Jupyter notebooks on Colab.
