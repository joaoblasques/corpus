# Model Routing - Choosing the Right Model for the Job

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-model-routing-choosing-the-right-model-for-the-job.md`

Model routing is a practice that dynamically selects the right AI model for a task, balancing cost, speed, and performance. This approach is more efficient and effective than relying on a single model for everything. Key tools and platforms for model routing include OpenRouter and Perplexity.

**Key topics**
- Model Routing
- AI Model Selection
- Multi-Model Systems
- Cost Optimization
- Latency Reduction
- Quality Improvement
