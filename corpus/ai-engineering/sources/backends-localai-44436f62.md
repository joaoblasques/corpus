# "Backends :: LocalAI"

> **Quick intake** (obsidian-list). [open source](https://localai.io/backends/index.html)

LocalAI supports various backends for running AI models, including LLMs, speech-to-text, text-to-speech, and more. Backends can be managed through the LocalAI web interface or by creating a backend gallery. The documentation provides information on creating and publishing a new backend.
