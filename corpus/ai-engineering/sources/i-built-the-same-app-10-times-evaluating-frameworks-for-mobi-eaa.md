# "I Built the Same App 10 Times: Evaluating Frameworks for Mobile Performance | Loren Stewart"

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-i-built-the-same-app-10-times-framework-evaluation.md`

The article evaluates the mobile performance of various web frameworks by building the same Kanban board application in each, highlighting the importance of bundle size and first contentful paint (FCP) times. Next-gen frameworks like Marko and SolidStart offer superior mobile performance. The article emphasizes the need for mobile-first design and selecting frameworks that prioritize long-term mobile performance.

- Mobile Performance
