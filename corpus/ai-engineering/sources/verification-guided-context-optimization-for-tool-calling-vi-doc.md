# verification-guided-context-optimization-for-tool-calling-vi

> **Quick intake** (web). [open source]("https://arxiv.org/abs/2512.13860")

A framework called Verification-Guided Context Optimization (VGCO) is proposed to refine tool-related documentation and knowledge base context for large language models. VGCO uses LLMs as editors to automatically refine documentation and context. It achieves significant improvements in accuracy, robustness, and generalization across LLMs.

**Key topics**
- Verification-Guided Context Optimization
- Tool Calling
- Hierarchical LLMs-as-Editors
- Large Language Models
- Documentation Refinement
- Knowledge Base Context
