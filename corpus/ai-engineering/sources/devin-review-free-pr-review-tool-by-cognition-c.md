# Devin Review — Free PR Review Tool by Cognition

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-devin-review-free-pr-tool-cognition.md`

Devin Review is a free PR review tool that replaces paid tools, saving $15-$25 per use. It offers features like autofix, smart diff organization, and codebase-aware chat. Users can activate it by replacing github.com with devinreview.com in any PR URL.
