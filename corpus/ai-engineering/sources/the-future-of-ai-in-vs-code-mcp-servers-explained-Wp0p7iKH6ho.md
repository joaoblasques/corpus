# "The Future of AI in VS Code: MCP Servers Explained!"

> **Quick intake** (YouTube · Burke Holland · playlist _Data Engineering_). [watch on YouTube](https://youtu.be/Wp0p7iKH6ho) · [transcript](../../../raw/youtube/youtube-Wp0p7iKH6ho-the-future-of-ai-in-vs-code-mcp-servers-explained.md)

The video explains the MCP (Model Context Protocol) servers in Visual Studio Code, which enable clients like GitHub Copilot to talk to external databases and services. It covers how to install and use MCP servers, specifically the Perplexity MCP server, to access web information and improve GitHub Copilot's knowledge. The key takeaway is that MCP servers can enhance the capabilities of AI-powered tools like GitHub Copilot.
