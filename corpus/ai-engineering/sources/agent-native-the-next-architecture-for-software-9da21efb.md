# "Agent-Native: The Next Architecture for Software"

> **Quick intake** (obsidian-list). [open source](https://www.builder.io/blog/agent-native-architecture)

Agent-native architecture combines human and AI agent interaction, enabling shared actions, data, permissions, and context. This approach removes the trade-off between polished interfaces and powerful agents. Agent-native applications are designed to be cloneable, with a shared action model and context awareness.

**Key topics**
- Agent-native architecture
- Human-AI collaboration
- Shared action model
- Context awareness
- Agent-native applications
- Cloneable apps
