# Source: Harvard CS50's AI with Python

**What it is**: Harvard's full university course "Introduction to Artificial Intelligence with Python," taught by Brian Yu — a comprehensive tour of classical and modern AI [^src1]. The canonical reference for the [[ai-engineering/ai-fundamentals|AI Fundamentals]] scaffold in this corpus.

## Curriculum (six units)

1. **Search** — DFS/BFS/greedy/A\*; adversarial search (minimax, alpha-beta) for games [^src1].
2. **Knowledge** — knowledge bases, propositional logic, model checking, inference [^src1].
3. **Uncertainty** — probability, Bayesian networks, Markov models [^src1].
4. **Optimization** — local search, linear programming, constraint satisfaction [^src1].
5. **Learning** — supervised/unsupervised/reinforcement learning; overfitting; gradient descent [^src1].
6. **Neural networks & language** — perceptrons → CNN/RNN; natural language processing [^src1].

Each unit ships hands-on Python projects to implement the algorithms [^src1].
