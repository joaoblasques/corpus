# Manage budgets and cost controls for Genie

> **Quick intake** (obsidian). raw stub: `notes-clippings-manage-budgets-and-cost-controls-for-genie.md`

This document explains how to manage budgets and cost controls for Genie, a pay-as-you-go model with a per-user free monthly allowance, and how to query Genie usage and cost. Budgets can be created and managed in the account console, and they track LLM usage, but not compute used to run queries. Alerts can be sent when thresholds are met, and usage can be blocked.
