# DevOps - DevSecOps CI Pipeline with GitHub Actions

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-devops-devsecops-ci-pipeline-with-github-actions.md`

This document discusses DevSecOps, a security-focused approach to DevOps that integrates security checks throughout the development lifecycle. It highlights the importance of shifting security left and using tools like GitHub Actions, SAST, and Docker Scout for vulnerability scanning. The key takeaway is that security should be integrated from the start to catch vulnerabilities before production.

- DevSecOps
- CI/CD pipeline
- GitHub Actions
