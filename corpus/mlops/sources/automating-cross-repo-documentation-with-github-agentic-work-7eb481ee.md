# Automating cross-repo documentation with GitHub Agentic Workflows

> **Quick intake** (obsidian-list). [open source](https://github.blog/ai-and-ml/github-copilot/automating-cross-repo-documentation-with-github-agentic-workflows/)

The document describes how to automate cross-repo documentation using GitHub Agentic Workflows, a project from the GitHub Next team. This approach enables the creation of SME-reviewed docs pull requests, closing the gap between release and documentation. The solution uses a GitHub App token with explicit repo and permission scopes to ensure security.
