# A Catalog Is All You Need

> **Quick intake** (obsidian-list). [open source](https://roundup.getdbt.com/p/a-catalog-is-all-you-need)

The author discusses the evolution of data infrastructure and the importance of catalogs in enabling interoperability across different query engines. They showcase a dbt demo that writes directly to a data platform's catalog, highlighting the shift towards a catalog-based unit of interoperability. The author also touches on the progress made in open data infrastructure and the role of Apache Iceberg in facilitating this shift.

**Key topics**
- dbt demo
- catalog-based interoperability
- Apache Iceberg
- open data infrastructure
- data platform catalogs
- Iceberg REST catalog
