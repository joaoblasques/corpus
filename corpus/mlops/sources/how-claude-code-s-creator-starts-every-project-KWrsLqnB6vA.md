# How Claude Code’s Creator Starts EVERY Project

> **Quick intake** (YouTube · Austin Marchese · playlist _Claude Code_). [watch on YouTube](https://youtu.be/KWrsLqnB6vA) · [transcript](../../../raw/youtube/youtube-KWrsLqnB6vA-how-claude-codes-creator-starts-every-project.md)

The video discusses how to effectively use Claude Code by starting every project in plan mode, using a Claude.md file, and keeping it minimal to avoid confusion. Boris Cherny's approach emphasizes planning and minimalism to get the model on track. The key takeaway is to do the minimal possible thing to get the model on track and add more instructions as needed.
