# DevOps - Docker Model Runner for Local LLMs

> **Quick intake** (obsidian). raw stub: `notes-03-resources-study-notes-devops-docker-model-runner-for-local-llms.md`

Docker Model Runner is a feature in Docker Desktop that enables running LLMs locally without CUDA drivers, providing GPU acceleration and OpenAI-compatible API. It supports various platforms and can be accessed programmatically. This feature is a better choice for those already using Docker heavily.
