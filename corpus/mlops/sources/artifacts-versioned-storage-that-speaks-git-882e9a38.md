# "Artifacts: versioned storage that speaks Git"

> **Quick intake** (obsidian-list). [open source](https://blog.cloudflare.com/artifacts-git-for-agents-beta/)

Artifacts is a versioned file system that speaks Git, designed for agents and scalable to meet the demands of large-scale systems. It provides a REST API and native Workers API for creating repositories, generating credentials, and commits. Artifacts is built on top of Durable Objects and uses a Git server implementation written in Zig, compiled to Wasm.
