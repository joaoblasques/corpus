# ADBC?

> **Quick intake** (obsidian-list). [open source](https://dataengineeringcentral.substack.com/p/adbc)

ADBC (Arrow Database Connectivity) is an open-source API standard for streaming data between applications and databases without row-by-row translation overhead. It leverages the Apache Arrow columnar memory format for efficient data transfer and access. ADBC provides a consistent API across languages and is backed by the Apache Foundation.

**Key topics**
- ADBC
- Apache Arrow
- columnar memory format
- data transfer
- database connectivity
- API standard
