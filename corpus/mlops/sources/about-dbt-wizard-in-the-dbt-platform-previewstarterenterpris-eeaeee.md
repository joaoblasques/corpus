# about-dbt-wizard-in-the-dbt-platform-previewstarterenterpris

> **Quick intake** (web). [open source]("https://docs.getdbt.com/docs/cloud/dbt-copilot")

dbt Wizard is a tool for governed data development in dbt that helps teams ship trusted dbt changes faster and with less risk. It uses native dbt metadata and validates with warehouse awareness. dbt Wizard supports different AI providers and is available in the dbt platform and as a standalone feature.
