# About MetricFlow | dbt Developer Hub

> **Quick intake** (obsidian-list). [open source](https://docs.getdbt.com/docs/build/about-metricflow)

MetricFlow is a SQL query generation tool that streamlines metric creation across different data dimensions for diverse business needs. It operates through YAML files and is developed and maintained by dbt Labs. MetricFlow simplifies the SQL process through metric YAML configurations.
