# Silent Data Loss Detection — Pipeline Volume Anomaly Monitoring with BigQuery and AI

> **Quick intake** (obsidian). raw stub: `notes-03-resources-articles-silent-data-loss-pipeline-volume-anomaly-detection-bigquery.md`

A BigQuery-native anomaly detection system monitors pipeline volume anomalies, detecting silent data loss and partial drops that standard pipeline health checks miss. The system uses built-in BigQuery capabilities, a single dbt model, and Google's TimesFM foundation model. It requires zero per-table configuration by default.
