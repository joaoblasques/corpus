# Apache Iceberg and the catalog layer (w/ Russell Spitzer)

> **Quick intake** (obsidian-list). [open source](https://roundup.getdbt.com/p/apache-iceberg-and-the-catalog-layer)

This article discusses Apache Iceberg, a high-volume analytic data file format, and its evolution through various versions. It covers the importance of the catalog layer and the role of Apache governance in maintaining the project. The article also touches on the Polaris project, an Apache incubator project that implements a catalog that speaks the Iceberg REST catalog spec.

**Key topics**
- Apache Iceberg
- catalog layer
- Apache governance
- Polaris project
- high-volume analytic data file format
- data lakehouse
