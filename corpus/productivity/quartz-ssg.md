# Quartz SSG

**TL;DR** — Quartz is a fast, batteries-included TypeScript static-site generator that publishes digital gardens and Markdown notes as websites, free to host. The primary open-source tool for Obsidian-vault publishing [^src1].

## What it is

- **Repository**: `jackyzha0/quartz` on GitHub — 12,577 stars (at collection date 2026-06-22) [^src1].
- **Language**: TypeScript [^src1].
- **Latest release**: v4.0.8 (per GitHub metadata); README is titled "Quartz v5" [^src1].
- **GitHub topics**: `digital-garden`, `networked-thought`, `obsidian`, `obsidian-md`, `quartz-ssg`, `static-site-generator` [^src1].
- **Documentation**: quartz.jzhao.xyz [^src1].
- **Community**: Discord server open to contributors and users [^src1].

## Core function

"A set of tools that helps you publish your digital garden and notes as a website for free" [^src1]. The repo description characterises it as "a fast, batteries-included static-site generator that transforms Markdown content into fully functional websites" [^src1].

The networked-thought and obsidian topics signal that WikiLink resolution and graph-based navigation are design goals [^src1]; the specific feature implementation (backlinks, graph view, search, RSS) is not detailed in the available source. [unsourced — see quartz.jzhao.xyz docs]

## Community and governance

Quartz maintains a Citizen Code of Conduct aimed at an inclusive contributor base regardless of background, with enforcement up to permanent expulsion for violations [^src1]. The project accepts GitHub Sponsors [^src1].

## Why it matters in the productivity domain

The `networked-thought` and `digital-garden` topics explicitly position Quartz as infrastructure for externalising a second brain — turning private Markdown notes into a navigable public or semi-public reference site [^src1]. It is the most-starred SSG in the `obsidian-md` GitHub topic category (as of ingest date) [^src1].
