Hello Jonas,

If you are in the data space, you might have heard of open table
formats such as Apache Iceberg, Apache Hudi, or Delta Lake. It
can be confusing; some questions people have are

* What exactly are Open Table Formats (OTF)? Is it more than just
a pointer to some metadata files that helps you sift through the
data quickly?
* What is the difference between table formats (Apache Iceberg,
Apache Hudi, Delta Lake) & file formats (Parquet, ORC)?
* How do OTFs work?

If you have any of the above questions (or want to stay
informed), this post is for you. Understanding the underlying
principle behind open table formats will enable you to deeply
understand what happens behind the scenes and make the right
decisions when designing your data systems.
